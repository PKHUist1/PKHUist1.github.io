Canadian Demographic Analytics Platform

A Microsoft Fabric analytics solution built to demonstrate end-to-end data engineering, analytics engineering and business intelligence capabilities using publicly available Canadian demographic and economic datasets.

The project implements a Medallion Architecture using Lakehouse storage, PySpark transformations, Semantic Models and Power BI reporting to deliver province-level demographic and economic insights.

Project Overview

This project was created to demonstrate a complete Microsoft Fabric analytics workflow from ingestion through reporting. Multiple public Canadian datasets were consolidated, transformed and enriched to create a single analytical model capable of supporting interactive reporting and exploration.

The solution combines demographic, economic and geographic information and enriches it with additional province metadata collected through web scraping.

The completed solution delivers a unified analytics platform that combines multiple Canadian demographic and economic datasets into a single reporting experience. The project demonstrates the full lifecycle of a modern Microsoft Fabric implementation, from ingestion and transformation through to semantic modelling and business intelligence.
