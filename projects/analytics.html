Canadian Demographic Analytics Platform

A Microsoft Fabric analytics solution built to demonstrate end-to-end data engineering, analytics engineering and business intelligence capabilities using publicly available Canadian demographic and economic datasets.

Project Overview

This project demonstrates a complete analytics solution built in Microsoft Fabric using publicly available Canadian demographic and economic datasets.

The solution combines data ingestion, transformation, enrichment, dimensional modelling, semantic modelling and reporting within a single Fabric environment while following a Bronze, Silver and Gold Medallion Architecture.

Power BI Reporting

Main dashboard providing province-level demographic and economic analysis.

Interactive reporting experience with filtering, drill-through and geographic insights.

Solution Architecture

Fabric Pipeline orchestrating Bronze, Silver and Gold notebook execution and managing the end-to-end workflow.

Data Engineering

The platform uses a Medallion Architecture implemented within a Fabric Lakehouse. PySpark notebooks perform cleansing, standardisation, enrichment and dimensional modelling.

Bronze and Silver Lakehouse structure used to organise raw and transformed datasets.

PySpark transformation logic used during ingestion, cleansing and standardisation.

Transformation and business-rule application between Bronze and Silver layers.

Semantic Modelling

Star-schema semantic model used to support analytical reporting and Power BI consumption.

Outcome

The completed solution delivers a unified analytical platform that combines demographic, economic and geographic information into a single reporting experience. The project demonstrates practical Microsoft Fabric skills across data engineering, modelling and business intelligence disciplines while showcasing an end-to-end analytics workflow.
