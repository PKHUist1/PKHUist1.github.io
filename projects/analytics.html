End-to-End Analytics Platform

End-to-end Microsoft Fabric analytics solution built using a Medallion Architecture, Spark notebooks, Semantic Models and Power BI.

Microsoft Fabric Lakehouse PySpark Data Pipelines Dataflow Gen2 Semantic Model Power BI

Overview

This project was built to demonstrate a complete analytics solution using Microsoft Fabric and publicly available Canadian demographic and economic datasets.

The solution combines data ingestion, transformation, dimensional modelling, semantic modelling and reporting within a single Fabric environment.

Key Features

  • Bronze, Silver and Gold Medallion Architecture
  • Automated data ingestion using Fabric Pipelines
  • PySpark transformation and cleansing
  • Wikipedia enrichment of province metadata
  • Dimensional modelling and semantic layer
  • Interactive Power BI reporting

Outcome

The final solution provides a single analytical view of Canadian demographic and economic data while demonstrating end-to-end analytics engineering practices within Microsoft Fabric.
