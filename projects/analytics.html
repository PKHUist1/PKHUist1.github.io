Canadian Demographic Analytics Platform

A Microsoft Fabric analytics solution built to demonstrate end-to-end data engineering, analytics engineering and business intelligence capabilities using publicly available Canadian demographic and economic datasets.

The project implements a Medallion Architecture using Lakehouse storage, PySpark transformations, Semantic Models and Power BI reporting to deliver province-level demographic and economic insights.

Project Overview

This project demonstrates a complete analytics solution built in Microsoft Fabric using publicly available Canadian demographic and economic datasets.

The platform combines ingestion, transformation, enrichment, dimensional modelling and reporting within a single Fabric environment while following a Bronze, Silver and Gold Medallion Architecture.

Data Engineering

The platform uses a Medallion Architecture implemented within a Fabric Lakehouse. PySpark notebooks perform cleansing, standardisation, enrichment and dimensional modelling.
