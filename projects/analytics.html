A Microsoft Fabric analytics solution built to demonstrate end-to-end data engineering, analytics engineering and business intelligence capabilities using publicly available Canadian demographic and economic datasets.

The project implements a Medallion Architecture using Lakehouse storage, PySpark transformations, Semantic Models and Power BI reporting to deliver province-level demographic and economic insights.

Project Overview

This project demonstrates a complete analytics solution built in Microsoft Fabric using publicly available Canadian demographic and economic datasets. The platform combines data ingestion, transformation, enrichment, semantic modelling and reporting within a single Fabric environment.

The solution follows a Medallion Architecture and incorporates automated pipelines, PySpark transformations and dimensional modelling to deliver an interactive analytical experience through Power BI.
