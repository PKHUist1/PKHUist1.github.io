AI Data Agent for Conversational Analytics

A Microsoft Fabric solution demonstrating how a governed semantic model can be combined with Fabric Data Agents to deliver self-service analytics through natural language interaction.

The project focuses on semantic engineering, metadata governance, business-friendly modelling and conversational analytics. Rather than relying on analysts to answer ad-hoc questions, business users can interact directly with enterprise data using natural language while still benefiting from governed metrics and centrally managed business logic.

Microsoft Fabric Semantic Models Fabric Data Agent Lakehouse DAX AI Analytics Metadata Governance Conversational BI

Project Overview

This project demonstrates a semantic-model-first analytics architecture built using Microsoft Fabric. Sales data from the Contoso retail dataset is ingested into a Fabric Lakehouse, transformed using Spark and surfaced through a governed semantic model.

The semantic model contains curated business terminology, calculation groups, metadata descriptions, perspectives, security roles and reusable DAX measures. Fabric Data Agent is then layered on top of the semantic model to enable natural language interaction with enterprise data.

Semantic Engineering

A major focus of this project was creating a semantic layer that could support both traditional reporting and AI-driven analytics experiences.

Star schema semantic model built using fact and dimension tables including Sales, Customer, Product, Promotion, Store and Calendar.

Business-friendly metadata and descriptions added to improve discoverability and provide contextual understanding for both users and AI experiences.

Security roles, perspectives, relationships and calculation groups used to create a governed enterprise semantic layer.

Outcome

The completed solution demonstrates how semantic modelling, governance and AI can be combined to create a scalable self-service analytics experience. By placing a curated semantic layer at the centre of the architecture, business users gain access to trusted metrics through both traditional reporting and conversational interfaces.

The project showcases Microsoft Fabric capabilities across data engineering, semantic engineering and AI-powered analytics while demonstrating how organisations can prepare their analytical estate for the next generation of natural language data experiences.
