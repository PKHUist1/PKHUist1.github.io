Alberta Grid Operations Monitoring

A Microsoft Fabric Real-Time Intelligence solution designed to monitor live Alberta electrical grid telemetry and weather data using streaming ingestion, Eventstream, Eventhouse, KQL analytics and operational dashboards.

The platform provides near real-time visibility into electricity demand, reserve capacity, grid health, telemetry freshness and environmental conditions that influence energy consumption.

Microsoft Fabric RTI Eventstream Eventhouse KQL Power BI Streaming Analytics Telemetry Engineering Operational Intelligence

Business Scenario

Traditional reporting focuses on historical analysis, whereas operational teams require visibility into events as they happen. This solution demonstrates how Microsoft Fabric Real-Time Intelligence can be used to monitor electrical grid operations in near real-time.

The platform combines live Alberta grid telemetry from AESO with weather telemetry to provide situational awareness, operational health monitoring and environmental correlation analytics.

  • Current grid demand monitoring
  • Generation capacity tracking
  • Reserve margin monitoring
  • Grid stress detection
  • Load ramp detection
  • Telemetry freshness monitoring
  • Weather impact analysis
  • Operational state classification

Telemetry Ingestion

A custom Fabric notebook continuously collects telemetry from external APIs and publishes events into the real-time streaming architecture.

Python telemetry producer responsible for collecting and publishing operational telemetry.

Custom Fabric environment configured with additional libraries used for external event publishing and telemetry integration.

Eventhouse Storage

Eventhouse provides scalable operational storage for streaming telemetry, enabling time-series analytics, historical event retention and real-time querying through KQL.

KQL Operational Analytics

Operational intelligence is implemented using Kusto Query Language (KQL), providing real-time calculations, aggregations and state monitoring.

Current-state telemetry calculations used to monitor reserve capacity and grid utilisation.

Time-series aggregation and rolling operational metrics used for dashboard reporting.

Weather telemetry enrichment and transformation used to correlate environmental conditions with grid demand.

Operational Dashboard

The operational dashboard provides a real-time monitoring experience inspired by observability platforms. It combines live telemetry, health indicators, utilisation metrics, weather telemetry and operational alerts into a single monitoring interface.

  • Grid utilisation versus reserve monitoring
  • Operational health indicators
  • Alert status monitoring
  • Rolling energy consumption analysis
  • Weather and load correlation
  • Telemetry freshness monitoring
  • Operational snapshot reporting
